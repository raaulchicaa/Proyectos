Mini Wordle Numérico<br>
<b>Descripción</b>

Este proyecto es una versión simplificada de Wordle, pero utilizando dígitos en lugar de letras.
El objetivo del jugador es adivinar una secuencia numérica generada aleatoriamente en un número limitado de intentos.

<b>Funcionamiento</b>

El usuario indica:

La cantidad de dígitos del número secreto.

El número de intentos disponibles.

Se genera una tabla con las filas correspondientes a cada intento.

El jugador introduce los números mediante:

El teclado numérico mostrado en pantalla.

El teclado físico del ordenador.

Al completar una fila, el usuario debe pulsar el botón "COMPROBAR" para validar el intento.

<b>Reglas</b>

Al comprobar un intento:

Verde: el dígito es correcto y está en la posición correcta.

Amarillo: el dígito existe en el número pero en otra posición.

Rojo: el dígito no forma parte del número secreto.

Si todos los dígitos son correctos, el jugador gana la partida. Si se agotan los intentos, el juego termina.

Reinicio

El botón "RESET" permite comenzar una partida nueva recargando el juego.

<b>Tecnologías utilizadas</b>

CSS
